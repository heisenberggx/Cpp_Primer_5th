## 练习1.14

> 对比for循环和while循环，两种形式的优缺点各是什么？

for循环，对循环变量的初始化和修改控制等都放在语句开头，形式简便。  
while循环，循环变量的初始化在循环语句之前，适用于条件控制循环。
